# Train-Route-Station-Traffic-Analysis
Business Intelligence Dashboard for Train Performance &amp; Station Traffic Analysis using Python and Power BI.

📊 Train Route & Station Traffic Analysis Dashboard
📌 Project Overview

This project focuses on Business Intelligence analysis of train operations data to generate actionable insights for stakeholders.

The dashboard analyzes:

1. Train performance

2. Station traffic

3. Journey duration

4. Route efficiency

5. Operational KPIs

🛠 Tools Used

* Python (Pandas, NumPy)

* Power BI

* Data Cleaning & Preprocessing

* DAX Measures

* Pivot Analysis

📈 Key Insights

* CST-Mumbai is the highest traffic station.

* Train 15909 & 16317 operate longest distance routes.

* Total network coverage: 52M distance units.

* Total operational stops: 186K.

* Average journey duration: 18.74 hours.

📂 Project Workflow

1. Data Cleaning using Python

2. Business Metrics Preparation

3. Pivot & KPI Calculation

4. Power BI Dashboard Creation

5. Business Insights Generation
